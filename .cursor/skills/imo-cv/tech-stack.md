# Stack Tecnológico - imo.cv

## Backend

- Framework: Django 5.0 + Django REST Framework
- Database: PostgreSQL 15 + PostGIS 3.4
- Cache: Redis 7.2
- Search: Elasticsearch 8.11 (future)
- Queue: Celery 5.3 + RabbitMQ 3.12
- Multi-tenant: django-tenants (schema separation)
- Storage: DigitalOcean Spaces (S3 compatible)
- Auth: JWT + OAuth2
- Python: 3.11+

## Frontend

- Framework: Next.js 14.2.0 (App Router)
- Language: TypeScript 5.4.0
- Styling: Tailwind CSS 3.4.0 + Headless UI
- State: Zustand 4.5.0
- Data: React Query (TanStack Query) 5.20.0
- HTTP: Axios 1.6.0
- Maps: Mapbox GL JS 3.1.0
- Animations: Framer Motion 11.0.0
- Forms: React Hook Form 7.50.0 + Zod 3.22.0
- Icons: @heroicons/react 2.1.0
- Charts: Recharts 2.12.0
- i18n: next-i18next 15.0.0

## Infraestrutura

- Hosting: DigitalOcean App Platform
- CDN: Cloudflare CDN
- Database: DigitalOcean Managed PostgreSQL
- Cache: DigitalOcean Managed Redis
- Storage: DigitalOcean Spaces
- Monitoring: Sentry + UptimeRobot
- CI/CD: GitHub Actions
- Domain: imo.cv

## Mobile (Future)
